# SimpleMailer
Simple and unbloated Mail library for PHP

Dependencies: Smarty (http://www.smarty.net/) or SimpleTpl (https://github.com/Mikjaer/SimpleTpl)

Project goals:

 - Warning-free and neat code
 - Small and durable solution
